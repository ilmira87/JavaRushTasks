HTML редактор с графическим интерфейсом.
В качестве библиотеки для создания графического интерфейса используется Swing.
В качестве архитектурного каркаса нашего приложения используется MVC модель.

Графический интерфейс представляет собой окно, в котором будет меню и панель с двумя вкладками.
На первой вкладке будет располагаться текстовая панель, которая отрисовывает html страницу,
на ней можно форматировать и редактировать текст страницы.
На второй вкладке будет редактор, который отображает код html страницы, в нем будут видны все используемые html теги,
а также можно менять текст страницы, добавлять и удалять различные теги.
